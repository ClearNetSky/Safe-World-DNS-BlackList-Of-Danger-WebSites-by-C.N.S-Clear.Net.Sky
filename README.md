## 🇬🇧 English Version

### 📋 About

**Safe World DNS BlackList** is a cybersecurity initiative by **C.N.S [Clear Net Sky]** in **S.O.W.A format**. This project provides regularly updated DNS blocklists to help protect individuals, families, and organizations from harmful online content.

**Goal:** Protect people from unpredictable and dangerous content on the Internet.

---

## ✨ Features

- 🔒 **Comprehensive Protection:** Blocks dangerous websites including:
  - Gore and violent content
  - Suicide-related websites
  - Pornographic content
  - Other harmful and disturbing content
- 🤖 **AI-Powered:** Automated scanning and verification by AI systems
- 👥 **Community Supported:** 20+ online moderators reviewing submissions
- 🔄 **Regular Updates:** Continuously updated with new threats
- 🌐 **Easy Integration:** Compatible with major DNS filtering solutions

---

## 📥 Download & Usage

### Direct Link (RAW)

```
https://raw.githubusercontent.com/ClearNetSky/Safe-World-DNS-BlackList-Of-Danger-WebSites-by-C.N.S-Clear.Net.Sky/main/blacklist/blacklist_suicidical_and_killing.txt
```

### How to Use

#### Option 1: Pi-hole
1. Open Pi-hole admin panel
2. Navigate to **Group Management** → **Adlists**
3. Add the RAW link above
4. Update gravity: `pihole -g`

#### Option 2: AdGuard Home
1. Open AdGuard Home dashboard
2. Go to **Filters** → **DNS blocklists**
3. Click **Add blocklist**
4. Paste the RAW link and save

#### Option 3: NextDNS
1. Log in to NextDNS
2. Go to **Security** → **Blocklists**
3. Add custom blocklist with the RAW link

#### Option 4: Manual DNS Server
Use the blocklist with BIND, Unbound, or other DNS servers by importing the list and configuring appropriate block responses.

  - Порнографический контент

## 📥 Загрузка и использование

### Прямая ссылка (RAW)

```
https://raw.githubusercontent.com/ClearNetSky/Safe-World-DNS-BlackList-Of-Danger-WebSites-by-C.N.S-Clear.Net.Sky/main/blacklist/blacklist_suicidical_and_killing.txt
```

### Как использовать

#### Вариант 1: Pi-hole
1. Откройте панель администратора Pi-hole
2. Перейдите в **Group Management** → **Adlists**
3. Добавьте ссылку RAW выше
4. Обновите gravity: `pihole -g`

#### Вариант 2: AdGuard Home
1. Откройте панель AdGuard Home
2. Перейдите в **Фильтры** → **DNS-списки блокировки**
3. Нажмите **Добавить список блокировки**
4. Вставьте ссылку RAW и сохраните

#### Вариант 3: NextDNS
1. Войдите в NextDNS
2. Перейдите в **Security** → **Blocklists**
3. Добавьте пользовательский список блокировки со ссылкой RAW

#### Вариант 4: Ручной DNS-сервер
Используйте список блокировки с BIND, Unbound или другими DNS-серверами, импортировав список и настроив соответствующие ответы блокировки.
